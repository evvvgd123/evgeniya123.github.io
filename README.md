# Персональное портфолио (визитная карточка) разработчика

Портфолио-сайт для демонстрации информации о разработчике, проектов и контактов.  
Сделано на Next.js и опубликовано на GitHub Pages.

## Ссылка на сайт
- https://evvvgd123.github.io/evgeniya123.github.io/

## Возможности (соответствие ТЗ)
- Главная страница с приветствием и кратким представлением
- Раздел «Обо мне» (резюме, навыки, образование/опыт)
- Раздел «Проекты» с карточками (описание, технологии, ссылки на демо/код)
- Раздел «Контакты» (прямые ссылки + форма/вариант связи)
- Навигация по якорям (меню)
- Адаптивная верстка (mobile / tablet / desktop)
- Базовая SEO-оптимизация (метаданные, семантическая разметка)
- Быстрая загрузка за счёт статической генерации (Next.js export)

## Технологии
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Git, GitHub
- GitHub Actions (деплой)
- GitHub Pages (хостинг)

## Установка и запуск локально
Требования: Node.js (LTS) и npm.

```bash
npm install
npm run dev
